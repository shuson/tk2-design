# tk2-design

> tookitaki react ui components in common

## Install

```bash
npm install --save tk2-design
```

## Usage

```jsx
import React, { Component } from 'react'

import {MyComponent } from 'tk2-design'

class Example extends Component {
  render () {
    return (
      <MyComponent />
    )
  }
}
```

## Example

Go inside example folder and run

```bash
yarn start
```
